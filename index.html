📖 Формат JSON (2 типа вопросов)

// Тестовый вопрос (варианты)
{
  "id": 1,
  "type": "choice",
  "question": "Текст вопроса?",
  "options": ["А", "Б", "В"],
  "correct": [1],
  "category": "Общее",
  "difficulty": "easy"
}

// Открытый вопрос (ввод текста)
{
  "id": 2,
  "type": "open",
  "question": "Введите ответ...",
  "answer": "правильный ответ",
  "category": "ОГЭ",
  "difficulty": "medium"
}

💡 Для открытых вопросов можно указать несколько допустимых ответов через массив "answers": ["вариант1", "вариант2"]. Регистр, пробелы и «ё» при проверке игнорируются.

ОТКРЫТЫЕ (type: "open"): поля "id", "question", "answer" (строка) ИЛИ "answers" (массив допустимых ответов). Общие поля: "category" (строка), "difficulty" ("easy"/"medium"/"hard"). ПРАВИЛА: 1.
